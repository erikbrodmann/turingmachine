Embedded Reber Grammar

Simulieren und visualisieren Sie eine Turing Maschine (TM) die die (ERG) mit so wenig Zuständen wie möglich entscheidet.

Die TM soll nicht nur einen DEA zur ERG Erkennung nachbilden.

Sie sollen eine einfachere Lösung finden, die mit einem DEA nicht realisierbar ist.
